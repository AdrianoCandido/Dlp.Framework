Installing the DLP Framework

Once configured the DLP NuGet Repository, the DLP Framework can be installed in your projects just like any other nuget package. To install the current version of DLP Framework in your project, just follow these steps.

  1. In Solution Explorer, right click the solution name and choose Manage NuGet Packages for Solution...
  2. In the Manage NuGet Packages dialog, click the DLP entry under the Online node;
  3. You shall see all the available packages from DLP;
  4. Find the package that you need and click the Install button;
  5. Select the projects where the packages must be installed and click OK.

The packages are going to be installed to the selected projects and are ready to be used.

Removing the DLP Framework

To remove the DLP Framework after you have added it to one or more projects, do as follow:

  1. In Solution Explorer, right click the solution name and choose Manage NuGet Packages for Solution...
  2. In the Manage NuGet Packages dialog, click the Installed packages node;
  3. Find the packages that you want to remove and click the Uninstall button;
  4. Select the projects from which the packages must be removed and click OK;

The packages are going to be removed from the selected projects.
