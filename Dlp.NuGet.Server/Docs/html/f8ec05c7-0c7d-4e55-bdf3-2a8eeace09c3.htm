Welcome to the DLP SDK

DLP SDK is a set of packages with selected services and methods elaborated to simplify and standardize simple everyday tasks. The methods provided by the DLP SDK were tested and approved, being used in real production applications, ensuring quality and safety. In addition to this, all the code are 100% covered by unit tests, giving you the security to focus exclusivelly on your business logic.

The DLP SDK is mainly consisted of extension methods, creating a transparent flow in the development process, without the need to instantiate several classes just for simple operations.

A set of connectors are also available to simplify the communication with databases, cache services, authentication and more.
