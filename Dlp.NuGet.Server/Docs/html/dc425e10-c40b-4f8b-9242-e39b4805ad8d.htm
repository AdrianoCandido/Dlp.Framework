Install DLP SDK Packages

This section describes how to add or remove DLP SDK Packages in your projects.

Prerequisites

Before proceeding, be sure to have the DLP Repository configured in your Visual Studio IDE. Check the Setting up the DLP Repository section if you have not configured your IDE.

Installing DLP SDK Packages

Once configured the DLP Repository, any DLP SDK Package can be installed in your projects just like any other NuGet package.

Adding DLP SDK Packages to one or more projects

  1. In Solution Explorer, right click the solution name and choose Manage NuGet Packages for Solution...

  2. In the Manage NuGet Packages dialog, click the DLP Repository entry under the Online node.

  3. You shall see all the available packages from DLP SDK.

  4. Find the package that you need and click the Install button.

  5. Select the projects where the packages must be installed and click OK.

The packages are going to be installed to the selected projects and are ready to be used.

Uninstall DLP SDK Packages

If you want to remove one or more DLP SDK Package after you have added, just proceed as any other Nuget Package uninstall.

Removing DLP SDK Packages from one or more projects

  1. In Solution Explorer, right click the solution name and choose Manage NuGet Packages for Solution...

  2. In the Manage NuGet Packages dialog, click the Installed packages node.

  3. Find the packages that you want to remove and click the Uninstall button.

  4. Select the projects from which the packages must be removed and click OK.
