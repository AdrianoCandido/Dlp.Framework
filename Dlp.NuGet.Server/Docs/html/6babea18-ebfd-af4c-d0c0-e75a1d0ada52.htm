Examples

The following example shows how to use the ExecuteReader method to get a list of rows.

C#
Copy
// Query to load transactions.
string query = @"SELECT TransactionId, AmountInCents FROM Transaction WHERE CreateDate >= @SearchStartDate AND CreateDate < @SearchEndDate;";

// Instantiate the database connector.
using (DatabaseConnector databaseConnector = new DatabaseConnector(connectionString)) {

  // Return a list of transactions, sorted by the CreateDate column in descending order.
  IEnumerable<Transaction> result = databaseConnector.ExecuteReader<Transaction>(query, new { SearchStartDate = searchStartDate, SearchEndDate = searchEndDate });
}

The following example shows how to load data with a collection as parameter for an 'IN' clause.

C#
Copy
// Query to load MerchantData.
string query = @"SELECT Name FROM Merchant WHERE MerchantKey IN (@MerchantKeyCollection)";

// A collection to be used as the query MerchantKeyCollection parameter.
IEnumerable<Guid> merchantKeyCollection = new Guid[] { Guid.Parse("fee2437e-c810-4c2b-a836-5f619f80bb76"), Guid.Parse("c5d66001-8a39-4a04-a22c-d3a190018c46") };

IEnumerable<MerchantData> result;

// Instantiate the database connector.
using (DatabaseConnector databaseConnector = new DatabaseConnector(connectionString)) {

  // Get the merchant data.
  result = databaseConnector.ExecuteReader<MerchantData>(query, new { MerchantKeyCollection = merchantKeyCollection });
}
