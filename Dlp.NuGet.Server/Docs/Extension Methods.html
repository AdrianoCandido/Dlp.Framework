Name Description

 CollectionExtensions Methods
Name Description
AsString(this IEnumerable source, char separator = ',', string surroundWith = null) Converte uma coleção em uma string, com cada elemento separado por vírgula, por padrão.
 DateTimeExtensions Methods
Name Description
ChangeTimeZone(this DateTime source, string sourceTimeZoneId, string targetTimeZoneId) Muda o fuso horário de uma data.
ToIso8601String(this DateTime source) Converte uma data para uma string no padrão ISO8061. Ideal para utilização no retorno de contratos em REST.
 ObjectExtensions Methods
Name Description
BinarySerializer(this object source) Converte o objeto especificado em um array de bytes.
Clone<T>(this T source) Clona um objeto, gerando uma nova instancia que não mantém referência ao objeto original.
