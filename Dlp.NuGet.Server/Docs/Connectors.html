Understanding the DLP Connectors

This section describes the available DLP Connectors and its uses. Having configured the DLP Repository in Visual Studio, you can install the connector searching for Dlp.Connectors in NuGet Package Manager.

As a general rule, connectors are utility classes that allow your application to communicate with external services, like database, cache service, authentication service, and so on.

At the moment, the following connectors are available.

DatabaseConnector

This is a simple, yet very powerful ORM. Based on Dapper, but going even further, the DatabaseConnector solves the most variable problems encountered during more than 2 years of development.

Some of the features are:

  • Native pagination of query results
  • Use of collections for clause "IN". For example: "SELECT Name FROM Merchant WHERE MerchantId IN (@MerchantIdCollection);"
  • Automatic mapping of database entities
  • Simplified usage of database transactions
  • BulkInsert from a collection
  • Dynamic types as parameter types. Create new types or use your existing database entities.

For a complete list of supported operations with samples, check de DatabaseConnector section.
