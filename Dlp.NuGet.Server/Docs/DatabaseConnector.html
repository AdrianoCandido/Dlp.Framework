Database Connector
Understanding the DLP Connectors > Database Connector

The DatabaseConnector is an easy and intuitive module that simplifies database operations. In general, most operations are performed with the already known ExecuteScalar, ExecuteReader and ExecuteNonQuery. The advantage comes when working with strong types where no mapping is necessary to retrieve the desired data.

The DatabaseConnector exposes the following methods.

 Constructors
Name Description
DatabaseConnector(int commandTimeoutInSeconds = 90) Inicializa uma nova instância do DatabaseConnector.
DatabaseConnector(string connectionString, int commandTimeoutInSeconds = 90) Inicializa uma nova instância do DatabaseConnector, com a string de conexão a ser utilizada para acessar o banco de dados.
 Properties
Name Description
CommandTimeoutInSeconds Obtém o timeout definido para as operações do connector, em segundos.
 Methods
Name Description
ExecuteScalar<T>(string query, dynamic parameters = null) Retorna a primeira coluna da primeira linha encontrada na consulta realizada.
ExecuteNonQuery(string query, dynamic parameters = null) Executa a query especificada no banco de dados e retorna a quantidade de registros afetados.
ExecuteReader<T>(string query, dynamic parameters = null) where T : new() Executa a leitura de informações do banco de dados e retorna os registros mapeados no objeto do tipo T.
ExecuteReader<T>(string query, int pageNumber, int pageSize, string orderByColumnName, SortDirection sortDirection, dynamic parameters = null) where T : new() Executa a leitura de informações do banco de dados, retornando o total de registros disponíveis e uma coleção com os dados relacionados na página especificada.
BulkInsert(string tableName, IEnumerable collection, SqlBulkCopyOptions sqlBulkCopyOptions = SqlBuilkCopyOptions.Default) Insere os dados de uma coleção na tabela especificada.
